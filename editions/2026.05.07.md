## 🛠️ Tools & Resources

### [Ossa: Towards the Next Generation Web](https://ossa.network/blog/post/2026/04/28/ossa-towards-the-next-generation-web)
"The Ossa Protocol is a secure, decentralized local-first synchronization protocol. The protocol lays the foundation for the next generation web where collaborative web applications are built on the decentralized web architecture previously described. In addition, the protocol is local-first since it realizes the vision proposed by Kleppmann et al. where users securely and collaboratively control their data."

### [Likewise: A protocol for decentralized personal knowledge graphs](https://getlikewise.ai/spec/)
"Likewise is a wire-level standard for decentralised personal knowledge graphs. A way for the things AI systems infer about a user (you're a frequent grocery shopper, you're close to Sarah, your commute is Tuesdays) to live in their hands rather than the platform that derived them. Append-only signed op log, hybrid logical clocks, capability-based UCAN delegation with sanitisation caveats, conditional inference auditing."

### [CTX](https://github.com/Alegau03/CTX#readme)
"CTX is a local runtime layer for OpenCode. It indexes the repository, stores reusable guidance as graph memory, exposes local MCP tools, generates OpenCode slash commands, and keeps local telemetry about token savings, cache reuse, and compact context packs."
