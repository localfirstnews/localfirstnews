## ✨ Apps

### [mu-txt](https://mutxt.com)
A polished, offline-first rich-text editor: floating menus, code blocks with syntax highlighting, embeds, time travel, and more out-of-the-box. Works in the browser at mutxt.com or as a desktop app via `npx mu-txt`. Documents are stored natively as `json-joy` Peritext CRDT, so forks can be merged cleanly with no central sync server. Open source on NPM as a React.js component `mutxt-react` and a Web component `mutxt-element`. [_json-joy / JSON CRDT / Peritext CRDT_]
